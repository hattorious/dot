## Project Notes

Dotfiles repo - key gotchas for future work:

1. Edit tool strips unicode powerline glyphs (U+E0B0 etc). Always use Python byte-level replacement for tmux.conf and plugins.vim.

2. lightline highlight groups are generated AFTER ColorScheme fires. Can't use autocmd to override tabsel color. Must edit the colorscheme file itself (rosepine_patched.vim).

3. lazygit doesn't support multi-key sequences like vim does. cc → had to map single key `c` to commitChangesWithEditor.

4. AGENTS.md is now the primary doc. CLAUDE.md just does @AGENTS.md. This is the right pattern for tool-agnostic docs.

5. XDG_CONFIG_HOME is set in runcom.zsh - all tools use ~/.config/, not ~/Library/.

6. zim's completion module calls compinit. Never call it manually before source ${ZIM_HOME}/init.zsh.

7. diffconflicts is incompatible with conflictstyle=diff3 - had to remove that setting.
